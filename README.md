WAD_AnalysisMatlab
==================

WAD IQC analyse modules in Matlab

For a new implementation of WADQC refer to https://bitbucket.org/MedPhysNL/wadqc

Note: version hosted in this repo is compatible with WAD2, and can be installed as a "factory" module in WAD2. For this purpose, a compiled version of this module is hosted at https://github.com/MedPhysQC/WADlab2_MR
